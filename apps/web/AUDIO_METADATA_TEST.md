# Тестирование аудио метаданных

## Что было сделано:

1. **Добавлено извлечение метаданных** из аудиофайлов:
   - Название трека (title)
   - Исполнитель (artist) 
   - Альбом (album)
   - Длительность (duration)
   - Обложка (coverArt)

2. **Создан минималистичный AudioPlayer** в стиле сайта:
   - Простые border и card компоненты
   - Компактная обложка 64x64px
   - Четкая типографика
   - Адаптивный дизайн

3. **Обновлен AttachmentUpload** для красивой превью:
   - Показ метаданных при загрузке
   - Минималистичный дизайн
   - Размер файла и длительность

## Как тестировать:

1. Откройте http://localhost:8083
2. Создайте новый тред или пост
3. Загрузите аудиофайл с метаданными (MP3 с тегами)
4. Проверьте консоль браузера - должны быть логи:
   ```
   Extracting metadata from: filename.mp3 audio/mpeg
   Parsed metadata: {title: "...", artist: "...", ...}
   Extracted metadata before transcoding: {...}
   ```

## Если метаданные не показываются:

Возможные причины:
- Файл не содержит ID3 тегов
- Файл в формате, который не поддерживается music-metadata
- Ошибка в консоли браузера

## Поддерживаемые форматы:
- MP3, FLAC, OGG, WAV, M4A, AAC
