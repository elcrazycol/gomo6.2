### B2. Swagger metadata + Security in `cmd/server/main.go`
```go
// @title           gomo6 API
// @version         1.0
// @description     gomo6 — open social platform API. Create bots, integrations, and apps.
// @termsOfService  https://gomo6.wtf/terms
// @contact.name    gomo6 Team
// @contact.url     https://gomo6.wtf
// @license.name    MIT
// @BasePath        /api/v1
// @schemes         https http
//
// @securityDefinitions.apikey BearerAuth
// @in                         header
// @name                       Authorization
// @description                Введите токен в формате: Bearer <ваш_токен>
```
The `@securityDefinitions` block enables the "Authorize" button in Scalar — devs paste token once, all "Try it out" requests include it automatically.

### B3. Swag annotations on key handlers (~45 functions)
Add `@Summary`, `@Tags`, `@Description`, `@Param`, `@Success`, `@Failure`, `@Router` to:

**Critical rules for all annotations:**
- **Path params**: Use `{id}` in `@Router`, NOT `:id`. Example: `// @Router /boards/{id} [get]`
- **Path params**: Always add `@Param id path string true "Description"` for each path param
- **Protected endpoints**: Add `// @Security BearerAuth` to every handler requiring auth
- **RPC handlers**: Use typed structs in `@Success` (NOT `gin.H` or `map[string]interface{}`). If handler returns raw JSON, create a response struct in `models/` or describe inline with `{object}`
- **time.Time fields**: Describe as `string` with `format:date-time` in annotations, swag doesn't auto-parse time.Time

### B4. Generate swagger.json
```bash

```
**NO `--parseDependency`** — fast build (<10 sec).

### B5. Serve swagger.json at `/api/v1/docs/json`
In `internal/api/routes/routes.go`, add:
```go

})
```
CORS handled by Caddy (already proxies `/api/*` for docs subdomain).

import "@scalar/api-reference/style.css";

const RestApiReference = () => {

    <div style={{ background: '#0f0f13', minHeight: '100vh', overflow: 'hidden' }}>

`overflow: 'hidden'` prevents Scalar's internal scrollbar from conflicting with the browser's native scrollbar during lazy chunk loading.
