3. `npx tsc --noEmit -p tsconfig.app.json` - no type errors
4. `npm run lint --workspace=@gomo6/web` - no lint errors
5. Commit and push per batch

After all batches:
- Final `npx vitest run --coverage` to confirm overall coverage improvement
- Verify CI typecheck passes
