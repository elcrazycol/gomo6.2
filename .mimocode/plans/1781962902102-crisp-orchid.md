The `protected` group already uses `AuthCacheMiddleware` which supports bot tokens (`gomo6_bot_*`). When a bot calls this endpoint, `claims.UserID` = bot's user ID, and `c.Get("is_bot")` = true. No additional middleware needed — the auth layer already handles bot vs user distinction.

**Security note**: The bot token is secret. Only bots with valid tokens can call this. The backend additionally validates cooldown at DB level, so even if someone leaks a bot token, they can't spam drops faster than 1/hour per user.

## Part 2: Dropsaway Bot — `/give` Command

### File: `/Users/lesha/codes/dropsaway/index.js`

```javascript
import { GomoBot } from '@gomo6/bot';

const bot = new GomoBot({ token: 'gomo6_bot_...' });

const cooldowns = new Map(); // userId -> expireTimestamp

const COOLDOWN_MS = 60 * 60 * 1000; // 1 hour
const DROP_AMOUNT = 15;

bot.on('ready', () => console.log('Бот запущен!'));

bot.on('message', async (ctx) => {
  if (ctx.text !== '/give') return;

  const userId = ctx.senderId;
  const now = Date.now();
  const expire = cooldowns.get(userId);

  if (expire && now < expire) {
    const remaining = Math.ceil((expire - now) / 60000);
    const word = remaining === 1 ? 'минуту' : remaining < 5 ? 'минуты' : 'минут';
    await ctx.reply(`Возвращайтесь через ${remaining} ${word}.`);
    return;
  }

  try {
    const res = await bot.api.request('POST', '/api/v1/bots/drops/give', {
      user_id: userId,
    });

    if (res.status === 429) {
      const remaining = Math.ceil(res.data.remaining_seconds / 60);
      const word = remaining === 1 ? 'минуту' : remaining < 5 ? 'минуты' : 'минут';
      await ctx.reply(`Возвращайтесь через ${remaining} ${word}.`);
      return;
    }

    cooldowns.set(userId, now + COOLDOWN_MS);
    await ctx.reply(`Вы получили ${DROP_AMOUNT} дропсов! Возвращайтесь через 1 час.`);
  } catch (err) {
    await ctx.reply('Ошибка при выдаче дропсов. Попробуйте позже.');
  }
});

bot.start();
```

**Key points:**
- Bot memory cooldown = fast path (no API call if known-cooldown)
- Backend DB cooldown = safety net (catches restarts, token leaks)
- When bot restarts → first `/give` after restart hits backend → backend checks DB → returns 429 if within hour → bot shows correct remaining time
