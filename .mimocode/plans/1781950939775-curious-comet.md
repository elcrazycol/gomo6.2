# Transaction Detail Sheet

## Что
При клике на транзакцию в истории кошелька — открывается минималистичная панель (Sheet снизу) с подробностями.

## Что показывать
- Тип операции (Перевод отправлен / Получен / Покупка / Подарок)
- Сумма (+ / -) крупно
- Баланс после операции
- Дата и время (полные)
- Описание (если есть)
- Для переводов: кому/от кого (username из reference_id)
- Для покупок: блокчейн + tx hash (если есть)
