# Разделение username и display_name + удаление фейковых почт

## Суть
- `username` — уникальный @handle, по которому ищут и логинятся. Можно менять.
- `display_name` — имя отображения, меняется свободно, не уникальное.
- Убрать генерацию `username@gomo6.local`. Email — опциональное поле.
- Login: username + пароль.

---

## 1. Миграция БД

**Файл:** `apps/backend-go/migrations/051_add_display_name_make_email_nullable.sql`

```sql
ALTER TABLE users ADD COLUMN display_name VARCHAR(255);
UPDATE users SET display_name = username WHERE display_name IS NULL;
ALTER TABLE users ALTER COLUMN email DROP NOT NULL;
ALTER TABLE users DROP CONSTRAINT IF EXISTS users_email_key;
-- UNIQUE на username уже есть из 001_initial_schema, регистрозависимый по умолчанию в PostgreSQL
```
