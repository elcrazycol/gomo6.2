# Drops (Капли) — мультичейн поддержка

## Текущее состояние

Реализовано: миграция, models, drops handler (balance, packages, config, callback, history), интеграция с gifts.go, фронтенд (DropsBalance, DropsShop, обновлены GiftCard/GiftSendDialog/GiftsTab/Profile).

**Проблема**: в `drops.go` поддерживается только ETH, Polygon, Solana. Нет Base. Токены захардкожены.

## Что нужно сделать

### 1. Добавить Base в `drops.go`

Добавить `DEPAY_RECEIVER_BASE` env и Base USDC токен:
- Base USDC: `0x833589fCD6eDb6E08f4c7C32D4f71b54bdA02913`
- Native ETH on Base: `0xEeeeeEeeeEeEeeEeEeEeeEEEeeeeEeeeeeeeEEeE`

### 2. Структурировать конфиг через env

Заменить захардкоженные адреса на конфигурируемые через env переменные:

```
DEPAY_RECEIVER_ETH=0x...
DEPAY_RECEIVER_POLYGON=0x...
DEPAY_RECEIVER_BASE=0x...
DEPAY_RECEIVER_SOLANA=...
```

Каждый chain добавляется в `accept` массив только если receiver задан.

### 3. Инструкция для пользователя (перед стартом)

#### Шаг 1: Создать кошелёк

Нужен один EVM кошелёк (один адрес работает на ETH, Polygon, Base) + Solana кошелёк:

**EVM (Metamask):**
1. Установить Metamask
2. Создать новый кошелёк / использовать существующий
3. Скопировать публичный адрес (0x...)

**Solana (Phantom):**
1. Установить Phantom
2. Создать кошелёк
3. Скопировать публичный адрес

#### Шаг 2: Настроить DePay Integration

1. Перейти на https://app.depay.com
2. Создать аккаунт / войти
3. Developers → Integrations → New Integration
4. Тип: **Payment Widget**
5. Название: gomo6-drops
6. **Callback URL**: `https://gomo6.wtf/api/v1/drops/callback`
7. **Domains**: `gomo6.wtf`
8. В разделе "Accepted tokens/blockchains" добавить:
   - Ethereum → USDC, ETH (native)
   - Polygon → USDC
   - Base → USDC, ETH (native)
   - Solana → USDC
9. В поле **Receiver** ввести свой адрес для каждого токена
10. Сохранить Integration
11. Скопировать **Integration ID** (нужен для фронта)
12. Скопировать **Public Key** (нужен для бэкенда)

#### Шаг 3: Сгенерировать RSA ключ pair

```bash
# На сервере или локально
openssl genpkey -algorithm RSA -out depay_private_key.pem -pkeyopt rsa_keygen_bits:2048
openssl rsa -pubout -in depay_private_key.pem -out depay_public_key.pem
```

Загрузить **содержимое `depay_public_key.pem`** в DePay Dashboard → Integration → Verify.

#### Шаг 4: Получить тестовые токены

Для тестирования на Base (низкие комиссии):

**Base Sepolia тестнет (для разработки):**
- Faucet: https://www.alchemy.com/faucets/base-sepolia
- Получить тестовые USDC и ETH

**Или Base mainnet с микросуммами:**
- Купить $1-2 USDC на бирже
- Отправить на свой адрес в Base сети
- DePay автоматически сконвертирует

#### Шаг 5: Настроить .env

```bash
# DePay
DEPAY_PUBLIC_KEY="-----BEGIN PUBLIC KEY-----\n...\n-----END PUBLIC KEY-----"
DEPAY_PRIVATE_KEY="-----BEGIN PRIVATE KEY-----\n...\n-----END PRIVATE KEY-----"

# Receiver addresses (один EVM адрес для ETH/Polygon/Base)
DEPAY_RECEIVER_ETH=0xВашАдрес
DEPAY_RECEIVER_POLYGON=0xВашАдрес
DEPAY_RECEIVER_BASE=0xВашАдрес
DEPAY_RECEIVER_SOLANA=ВашSolanaАдрес

# Frontend
VITE_DEPAY_INTEGRATION_ID=ваш-integration-id
```

#### Шаг 6: Тестирование

1. Запустить `docker compose up -d`
2. Открыть приложение, войти в профиль
3. Нажать "Купить капли"
4. Выбрать пакет (Стартовый $1.99)
5. DePay Widget откроется → подключить Metamask/Phantom
6. Оплатить из Base сети (низкие комиссии)
7. После подтверждения → баланс обновится
