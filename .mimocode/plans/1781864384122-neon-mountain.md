# gomo6 Bot SDK — Implementation Plan

## Суть
TypeScript NPM-пакет `@gomo6/bot` для написания ботов. Три компонента: HTTP Client (axios), WebSocket Client (ws), EventEmitter (GomoBot). Дополнительно: добавить поддержку бот-токенов в WebSocket handler на бэкенде.

---

## Шаг 0: Бэкенд — поддержка бот-токенов в WebSocket

**Проблема**: `apps/backend-go/internal/websocket/client.go` метод `handleAuth` вызывает `authService.ValidateToken()` который понимает только JWT. Бот-токены (`gomo6_bot_*`) отвергаются.

**Решение**: В `handleAuth` добавить проверку на `gomo6_bot_` префикс. Если токен начинается с `gomo6_bot_` — хэшировать SHA-256, искать в `bots` таблице, устанавливать `claims` с `UserID = bot_user_id`.

**Файл**: `apps/backend-go/internal/websocket/client.go`

```go
func (c *Client) handleAuth(data []byte) {
    // ... parse auth message ...
    
    // Try bot auth first
    if strings.HasPrefix(authMsg.Data.Token, "gomo6_bot_") {
        // SHA-256 hash → lookup in bots table → set claims
        // On success: c.authenticated = true, send "connected", subscribe to rooms
        return
    }
    
    // Existing JWT auth...
    claims, err := c.hub.authService.ValidateToken(authMsg.Data.Token)
    // ...
}
```

Также нужно добавить `db *sql.DB` в `Hub` struct и прокинуть из `main.go`.

**Авто-подписка**: После успешной авторизации бота в WebSocket, автоматически подписывать его на комнату `notifications_<bot_user_id>` (аналогично тому, как это делается для обычных пользователей). Это обеспечит доставку chat_message событий без ручной подписки.

## Шаг 4: WebSocket Client — `src/client/wsClient.ts`

```typescript
class WsClient extends EventEmitter {
  private ws: WebSocket | null;
  private token: string;
  private url: string;
  private reconnect: boolean;
  private reconnectInterval: number;
  private maxReconnectAttempts: number;
  private reconnectAttempts: number = 0;
  private pingInterval: NodeJS.Timeout | null = null;
  private subscribedRooms: Set<string> = new Set();
  
  connect(): void {
    this.ws = new WebSocket(this.url);
    this.ws.on('open', () => this.onOpen());
    this.ws.on('message', (data) => this.onMessage(data));
    this.ws.on('close', () => this.onClose());
    this.ws.on('error', (err) => this.emit('error', err));
  }
  
  private onOpen(): void {
    // Send auth immediately
    this.send({ type: 'auth', data: { token: this.token }, timestamp: Date.now() });
    // Start ping every 30s
    this.pingInterval = setInterval(() => this.ping(), 30000);
    // Reset reconnect counter
    this.reconnectAttempts = 0;
  }
  
  private onMessage(raw: Buffer | string): void {
    const msg = JSON.parse(raw.toString());
    
    switch (msg.type) {
      case 'connected':
        // Re-subscribe to rooms
        for (const room of this.subscribedRooms) {
          this.subscribe(room);
        }
        this.emit('connected', msg.data);
        break;
      case 'new_post': case 'new_thread': case 'new_reply':
      case 'like': case 'unlike':
      case 'new_chat_message': case 'message_edited': case 'message_deleted':
      case 'new_wall_post': case 'update_wall_post': case 'delete_wall_post':
      case 'new_notification':
      case 'user_online': case 'user_offline':
      case 'typing': case 'chat_typing': case 'read_receipt':
        this.emit('raw', msg);
        this.emit(msg.type, msg);
        break;
      case 'pong': break;
      case 'error': this.emit('error', new Error(msg.data?.error)); break;
    }
  }
  
  private onClose(): void {
    clearInterval(this.pingInterval!);
    this.emit('disconnected');
    if (this.reconnect) this.scheduleReconnect();
  }
  
  private scheduleReconnect(): void {
    if (this.reconnectAttempts >= this.maxReconnectAttempts) {
      this.emit('error', new Error('Max reconnect attempts reached'));
      return;
    }
    this.reconnectAttempts++;
    this.emit('reconnecting', this.reconnectAttempts);
    setTimeout(() => this.connect(), this.reconnectInterval);
  }
  
  subscribe(room: string): void {
    this.subscribedRooms.add(room);
    this.send({ type: 'subscribe', data: room, timestamp: Date.now() });
  }
  
  unsubscribe(room: string): void {
    this.subscribedRooms.delete(room);
    this.send({ type: 'unsubscribe', data: room, timestamp: Date.now() });
  }
  
  send(data: object): void {
    if (this.ws?.readyState === WebSocket.OPEN) {
      this.ws.send(JSON.stringify(data));
    }
  }
  
  disconnect(): void {
    this.reconnect = false;
    clearInterval(this.pingInterval!);
    this.ws?.close();
  }
}
```

## Шаг 5: Context Objects

### MessageContext — для chat_message events
```typescript
class MessageContext {
  constructor(
    private http: HttpClient,
    private raw: RawChatMessage
  ) {}
  
  get text(): string { return this.raw.content; }
  get conversationId(): string { return this.raw.conversation_id; }
  get senderId(): string { return this.raw.sender_user_id; }
  get messageId(): string { return this.raw.id; }
  get isEdited(): boolean { return this.raw.is_edited; }
  get sentAt(): Date { return new Date(this.raw.sent_at); }
  
  async reply(content: string): Promise<Message> {
    return this.http.sendMessage(this.raw.conversation_id, content);
  }
  
  async edit(content: string): Promise<void> {
    return this.http.editMessage(this.raw.conversation_id, this.raw.id, content);
  }
  
  async delete(): Promise<void> {
    return this.http.deleteMessage(this.raw.conversation_id, this.raw.id);
  }
}
```

### PostContext — для post_created events
```typescript
class PostContext {
  constructor(
    private http: HttpClient,
    private raw: RawPost
  ) {}
  
  get text(): string { return this.raw.content; }
  get threadId(): string { return this.raw.thread_id; }
  get postId(): string { return this.raw.id; }
  get userId(): string { return this.raw.user_id; }
  
  async reply(content: string): Promise<Post> {
    return this.http.createPost({ thread_id: this.raw.thread_id, content });
  }
}
```

## Шаг 6: Главный класс — `src/GomoBot.ts`

```typescript
import { EventEmitter } from 'events';

class GomoBot extends EventEmitter<BotEvents> {
  private http: HttpClient;
  private ws: WsClient;
  private config: BotConfig;
  
  constructor(config: BotConfig) {
    super();
    this.config = {
      baseUrl: 'https://gomo6.wtf',
      wsUrl: 'wss://gomo6.wtf/ws',
      reconnect: true,
      reconnectInterval: 3000,
      maxReconnectAttempts: 10,
      ...config,
    };
    this.http = new HttpClient(this.config.token, this.config.baseUrl!);
    this.ws = new WsClient(this.config.token, this.config.wsUrl!, this.config);
    
    // Wire up WS events → GomoBot events
    this.ws.on('connected', () => this.emit('ready'));
    this.ws.on('raw', (msg) => this.routeEvent(msg));
    this.ws.on('error', (err) => this.emit('error', err));
    this.ws.on('disconnected', () => this.emit('disconnected'));
    this.ws.on('reconnecting', (n) => this.emit('reconnecting', n));
  }
  
  get api(): HttpClient { return this.http; }
  
  start(): void {
    this.ws.connect();
  }
  
  stop(): void {
    this.ws.disconnect();
  }
  
  // Subscribe to specific rooms
  subscribeToThread(threadId: string): void {
    this.ws.subscribe(threadId);
  }
  
  subscribeToBoard(boardId: string): void {
    this.ws.subscribe(`board_${boardId}`);
  }
  
  subscribeToFeed(): void {
    this.ws.subscribe('feed');
  }
  
  subscribeToChat(conversationId: string): void {
    this.ws.subscribe(`chat_${conversationId}`);
  }
  
  private routeEvent(msg: any): void {
    switch (msg.type) {
      case 'new_chat_message':
        this.emit('message', new MessageContext(this.http, msg.data));
        break;
      case 'new_post':
      case 'new_reply':
        this.emit('post_created', new PostContext(this.http, msg.data));
        break;
      case 'new_thread':
        this.emit('thread_created', msg.data);
        break;
      case 'like':
        this.emit('like', msg.data);
        break;
      case 'unlike':
        this.emit('unlike', msg.data);
        break;
      // ... other events
    }
  }
}
```

**Важно**: `tsconfig.build.json` должен иметь `"declaration": true` в `compilerOptions`, чтобы в `dist/` генерились `.d.ts` файлы для автокомплита в VS Code.

**EventEmitter**: Использовать `import { EventEmitter } from 'node:events'` с дженериками. Требуется `@types/node >= 20` для полноценной типизации `EventEmitter<BotEvents>`. Если будут проблемы с типами — объявить `on`/`emit` через интерфейс `TypedEmitter`.
