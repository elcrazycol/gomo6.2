## Как использовать

1. **Загрузка нового аватара**: При загрузке через профиль, аватар автоматически добавляется в историю

2. **Просмотр истории**: Кликните на аватар в профиле - откроется галерея со всеми предыдущими аватарами

3. **Навигация**:
   - Стрелки влево/вправо
   - Свайпы на мобильных
   - Клик по миниатюрам внизу

4. **Удаление** (только для владельца):
   - Кнопка корзины в правом верхнем углу
   - Подтверждение удаления
   - При удалении текущего аватара, предыдущий становится активным

## Технические детали

### API эндпоинты
- `GET /rpc/v1/get_avatar_history?user_uuid=<uuid>` - получить историю
- `POST /rpc/v1/delete_avatar_from_history` - удалить аватар
  ```json
  {
    "avatar_id": "uuid",
    "requesting_user_id": "uuid"
  }
  ```

### База данных
```sql
-- Структура таблицы
CREATE TABLE avatar_history (
    id UUID PRIMARY KEY,
    user_id UUID REFERENCES users(id) ON DELETE CASCADE,
    avatar_url TEXT NOT NULL,
    uploaded_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
    is_current BOOLEAN DEFAULT FALSE
);
```
